Surgimento da internet

(Voltar)

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML siginifica HiperText Markup Language, traduzido ao potuguês: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar siginificado e organizar as informações de uma página na web. Sem isso o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens, vídeos, documentos, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e guardando informações.

Como funciona o HTML

(Voltar)

Através de um documento HTML, ou seja, um documento com a extensão .html ou .htm, o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá-lo. Os arquivos .html podem ser visualizados em qualquer navegador (como Google Chrome, Safari ou Mozilla Firefox).

Geralmente um site é composto por diversas páginas HTML, como por exemplo: um website que contenha três páginas (uma homepage, uma página de contato e uma página de produtos) receberá ao menos três documentos .html distintos, sendo uma para cada página do website.

O código pode ser escrito através de qualquer editor de texto, como o próprio bloco de notas. Cada página consiste em uma série de tags (também chamados de elementos) que podem ser considerados os blocos de construção das páginas. Portanto, esses blocos são a maneira com o qual o HTML faz a marcação dos conteúdos, criando uma hieraquia e a estrutura do mesmo, dividindo entre seções, parágrafos, cabeçalhos e outros.

O que são Tags

(Voltar)

Através de qualquer editor de texto, como o Sublime Text, o NotePad++ ou até mesmo o bloco de notas, é possível criar um documento com extensão .html que será renderizado pelos navegadores.

Conforme explicado anteriormente, este documento consiste em uma séria de tags. As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho, a fonte da letra, as cores, as quebras de linha e etc. A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fechamento, como a div. Há também tags de estrutura única, como a tag br que realiza uma quebra de linha.

História dos computadores

(Voltar)

A história dos computadores começou no século XX, com máquinas grandes e lentas que realizavam cálculos simples. O primeiro computador eletrônico foi o ENIAC, criado em 1946. Com o tempo, os computadores evoluíram, ficando menores, mais rápidos e acessíveis, até se tornarem parte essencial do nosso cotidiano.

Hoje, eles estão em todos os lugares: em casas, escritórios, bolsos (como smartphones), carros e até eletrodomésticos. Essa revolução tecnológica impactou profundamente a forma como vivemos, trabalhamos e nos comunicamos.

O que são Clients?

(Voltar)

Um client é qualquer dispositivo ou software que acessa um serviço em um servidor. Por exemplo, quando você abre um site no navegador, seu computador ou celular está funcionando como um cliente. O navegador envia uma solicitação ao servidor e aguarda uma resposta com o conteúdo do site.

Clients podem ser navegadores, aplicativos móveis, softwares de email ou qualquer outra aplicação que consuma serviços da web.

O que são Servers?

(Voltar)

Um servidor (ou server) é um computador especializado em fornecer dados, serviços ou recursos para outros dispositivos, chamados de clients. Ele fica "escutando" as requisições dos clients e responde com o conteúdo necessário — por exemplo, o HTML de uma página, uma imagem ou os dados de um banco de dados.

Servidores geralmente possuem alta capacidade de processamento e armazenamento, pois precisam lidar com muitas requisições simultâneas.

Como é uma estrutura básica de um HTML?

(Voltar)

Um documento HTML básico possui uma estrutura padrão com as seguintes tags principais: <!DOCTYPE html>, <html>, <head>, <title> e <body>. A tag <head> contém informações sobre o documento, como o título e links para estilos. A tag <body> contém todo o conteúdo visível da página.

Esse modelo é importante pois garante que os navegadores interpretem corretamente o conteúdo e apresentem a página de forma padronizada.

Entendendo a diferença entre listas ordenadas e não ordenadas

(Voltar)

Listas ordenadas são aquelas em que a ordem dos itens importa e são numeradas automaticamente pelo navegador. São criadas com a tag <ol>. Já as listas não ordenadas não seguem uma ordem específica e são representadas com marcadores (pontos), usando a tag <ul>.

Por exemplo, uma receita pode usar uma lista ordenada para indicar a sequência dos passos, e uma lista não ordenada para listar os ingredientes.

h1 até h6

(Voltar)

As tags <h1> até <h6> são usadas para títulos e subtítulos. O <h1> é o título mais importante e o <h6> o menos importante. Essa hierarquia ajuda a organizar o conteúdo da página e também melhora o SEO (Search Engine Optimization).

p

(Voltar)

A tag <p> é utilizada para definir parágrafos de texto. É uma das mais usadas em HTML e serve para separar e organizar os blocos de informação.

mark

(Voltar)

A tag <mark> é usada para destacar partes do texto, geralmente com um fundo amarelo. Pode ser útil para chamar atenção a termos importantes.

small

(Voltar)

A tag <small> exibe o texto em um tamanho menor, sendo comum para notas de rodapé ou informações complementares.

i

(Voltar)

A tag <i> é usada para colocar o texto em itálico. É usada para destacar termos, nomes científicos, estrangeirismos ou ênfases.

u

(Voltar)

A tag <u> sublinha o texto. Ela era mais comum no passado, mas hoje é usada com mais cuidado para não confundir com links.

strong

(Voltar)

A tag <strong> serve para destacar partes importantes do texto, deixando-o em negrito e dando significado semântico de importância.

ol

(Voltar)

<ol> é a tag que cria listas ordenadas (numeradas). Cada item dentro da lista é adicionado com a tag <li>.

ul

(Voltar)

<ul> é a tag usada para criar listas não ordenadas (com marcadores). Também utiliza a tag <li> para os itens.

li

(Voltar)

<li> é a tag que representa cada item dentro de uma lista, seja ordenada (<ol>) ou não ordenada (<ul>).

a

(Voltar)

A tag <a> é usada para criar links. O atributo href define para onde o link irá direcionar. Exemplo: <a href="https://www.google.com">Google</a>.

hr

(Voltar)

A tag <hr> insere uma linha horizontal na página, usada para separar visualmente seções de conteúdo.

sub

(Voltar)

A tag <sub> é usada para exibir texto em subscrito, ou seja, abaixo da linha do texto, como em fórmulas químicas: H2O.

sup

(Voltar)

A tag <sup> exibe o texto em sobrescrito, acima da linha normal, como em potências matemáticas: x2.

blockquote

(Voltar)

A tag <blockquote> serve para destacar uma citação longa. O texto fica com recuo, indicando que foi retirado de outra fonte.

font

(Voltar)

A tag <font> era usada para definir cor, tamanho e tipo da fonte, mas hoje é considerada obsoleta. Em vez dela, usa-se CSS para estilização.

del

(Voltar)

A tag <del> indica que um trecho de texto foi removido ou está obsoleto. O conteúdo aparece com um traço atravessando, como este exemplo.

abbr

(Voltar)

A tag <abbr> define uma abreviação. Ao passar o mouse sobre ela, aparece o significado completo. Exemplo: CSS.
